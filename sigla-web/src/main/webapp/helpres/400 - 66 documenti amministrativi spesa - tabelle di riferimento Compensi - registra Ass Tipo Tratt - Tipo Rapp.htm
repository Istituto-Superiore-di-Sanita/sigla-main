Compenso

 

Creazione e modifica di una associazione Tipo Rapporto/ Tipo Trattamento

 

Obiettivo

 

La scheda illustra la modalità per l’inserimento di una associazione Tipo Rapporto/ Tipo Trattamento.

 

Sviluppo

 

Al momento di richiamare  la funzione Documenti amministrativi – Compensi –  tabelle di riferimento – associazione Tipo Rapporto/ Tipo Trattamento –  gestione, il  sistema presenta il seguente pannello:

 

 

Tale tabella descrive le associazioni tra le tipologie di rapporto e le tipologie di trattamento previste per la gestione dei compensi.

Dal momento che per ogni soggetto anagrafico è possibile associare uno o più tipologia di rapporto si rende possibile la definizione delle tipologie di trattamento ammesse per una data anagrafica.

 

I dati che devono essere inseriti sono:

-                Rapporto: è il codice del rapporto, se conosciuto può essere inserito direttamente altrimenti è possibile trovarlo tramite i tasti di ricerca o ricerca guidata presenti alla destra del campo. Il sistema attinge i dati dalla tabella: Anagrafica clienti/fornitori – tabelle riferimento –  tipi di rapporto.

-                Trattamento: è il codice del trattamento, se conosciuto può essere inserito direttamente altrimenti è possibile trovarlo tramite i tasti di ricerca o ricerca guidata presenti alla destra del campo. Il sistema attinge i dati dalla tabella: Documenti amministrativi – Compensi –  tabelle di riferimento – tipo trattamento.

 

Terminato l’inserimento dei dati è possibile salvare.
